///ブックマークアプリ///
file inputからjsonファイルを読み込むことでリンク一覧を表示させることができる。
jsonファイルにurl,image,textを入力することで新たにリンクの追加が可能

ウェブブラウザにあるブックマークを参考に
画像を追加することで一目でどのリンクかわかる
より扱いやすいブックマークアプリ

//動作確認//
index.htmlをウェブブラウザで読み込み(chrome推奨)
file inputボタンをクリック
動作確認用のtestdata.jsonを読み込む
ブラウザにリンク一覧が表示されます。
